# Project 17: Pose Estimation

## Overview
CNN-based human pose estimation for detecting 17 body keypoints in images and videos.

## Key Features
- 17 keypoint detection
- Real-time inference
- Multi-person support
- Action recognition ready

## Use Cases
- Fitness apps
- Sports analytics
- Animation/motion capture
- AR/VR
- Healthcare

## Training
```bash
python train.py
```

## Performance
- MAE: <5 pixels
- Speed: 20+ FPS
