## üöÄ Quick Start

### Prerequisites
```bash
Python 3.8+
16GB RAM (32GB for LLM projects)
GPU optional (recommended for projects 23-43)
```

### Installation
```bash
# Clone repository
git clone https://github.com/YOUR_USERNAME/ml-automotive-portfolio.git
cd ml-automotive-portfolio

# Create virtual environment
python -m venv venv
venv\Scripts\activate  # Windows
# source venv/bin/activate  # Linux/Mac

# Install dependencies
pip install -r requirements.txt

# Validate portfolio
python validate_portfolio.py
```

### Run Your First Project
```bash
# Start with predictive maintenance
cd project_01_predictive_maintenance
python train.py

# Expected output: Trained model with 95%+ accuracy
```
